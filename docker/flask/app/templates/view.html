Welcome to the Churchill Database

Edit:

For the time being, only a small number of people have direct access to the database. Log in here if you are one of these people. If this is not the case, you can still provide suggestions, which will most likely be approved.

Why?

This website offers a funny way to visualize the relationships among people within our school. The data is stored in a graph database, which organizes data into nodes (such as people) and edges (representing relationships). This website was created simply because mapping the relationships among people in our school seemed like an interesting way to learn about graphs.

However, by putting this online, some considerations have arrised. It is important to note that the actual data itself was not given much consideration, so it should not be taken too seriously. Nevertheless, we believe that as different people contribute to this database, it will become more interesting and accurate. Our intention is not to cause any unnecessary drama. If there is any information about you that you are uncomfortable with being shared here, please inform us, and we will promptly remove it without further questions (although if it's here, it's likely already common knowledge within the school). We welcome any suggestions or feedback you may have.
